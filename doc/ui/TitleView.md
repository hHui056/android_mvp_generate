##### TitleView
可作为App通用的顶部bar，提供了以下功能

1、在左侧/右侧添加图片按钮/文字按钮

2、设置标题
###### xml属性

|  属性名 | 描述   |
|---|----|
| title  | 标题 |
| isSupportBack  | 是否显示左侧返回按钮 |
| rightImg1  | 右侧图片 |
| rightImg2  | 右侧第二张图片 |
| backImg  | 左侧返回按钮图片 |
| titleRightText  | 右侧文字 |
| titleBackgroundColor  | 标题栏背景颜色 |
| titleTextColor  | 标题文字颜色 |
| rightTextColor  | 右侧文字颜色 |

###### 事件

|  事件名称 | 描述   |
|---|----|
| titleRightImage1ClickListener  | 右边图片1点击事件监听 |
| titleRightImage2ClickListener  | 右边图片2点击事件监听 |
| titleRightTextClickListener  | 右边文字点击事件监听 |
