#### TableView
一般用于底部Table页导航切换，提供了以下功能

1、选中和非选中状态切换

2、图片和文字设置

##### xml属性

|  属性名 | 描述   |
|---|----|
| table  | 标题 |
| normalImg  | 常规状态（未选中）显示图片 |
| choiceImg  | 选中状态图片 |
| tableViewSelectedTextColor  | 选中时文字颜色 |
##### 主要方法

|  方法名 | 描述   |
|---|----|
| isChoice  | 设置选中状态 |
| onlyImageChangeChoice  | 选中时只修改图片不改变文字颜色 |
